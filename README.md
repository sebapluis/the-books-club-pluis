Este proyecto busca simular un ecommerce de libros, donde el usuario pueda tener una vista general del catalogo, como asi poder filtrar los libros por categorias (novelas, infantiles, historia), entrar en el libro que desee y poder leer una descripcion detallada, agregar las cantidades que desee (sin superar el stock disponible) al carrito en el cual le aparecera el precio final de su compra pudiendo desde el mismo quitar unidades como asi vaciar el carrito entero o finalizar su compra.
Elegida esta ultima opcion se le redirigir√° al checkout donde debera ingresar sus datos, realizada dicha accion con exito se le informara que su compra se realizo con exito y se le mostrata el codigo de la misma creado de forma automatica por firebase. 
Para darle estilo a la pagina utilize bootstrap.
